**Title:** Study label

|              |          |
| ------------ | -------- |
| **Type**     | `string` |
| **Required** | Yes      |
| **Format**   | `regex`  |

**Description:** A short label for the study. It should have no spaces and should be fewer than 50 characters.

**Examples:** 

```json

```

